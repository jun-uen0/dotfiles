### Setting up git alias
For macOS

~/.zshrc

1. Copy git/gitconfig
2. Run `cd` to move the home directory
3. Run `touch .gitconfig` to make a file
4. Run `vi .gitconfig` to enter the file
5. Paste to copy in the file
6. Save and exit
